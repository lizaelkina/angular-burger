# Stellar Burgers

- Pet-проект Stellar Burgers, переписанный на Angular.

## Описание проекта

- Веб-приложение бургерной со следующей функциональностью: главной страницей, личным
  кабинетом и лентой заказов.

## Используемые технологии

- Angular
- CCS3
- HTML5
